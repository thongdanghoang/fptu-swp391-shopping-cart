http://localhost:9090/oauth/authorize?response_type=code&client_id=client&scope=n3tk

Request access token by exchanging code
curl -v -XPOST -u client:secret "http://localhost:9090/oauth/token?grant_type=authorization_code&scope=n3tk&code="

use access token to access resource
curl -H "Authorization:Bearer " http:// localhost:9092/hello
